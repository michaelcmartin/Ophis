Getting a copy of Ophis

As of version 2.2, the Python Package Index offers Ophis at https://pypi.org/manage/project/ophis-asm/. This version may be installed on any system where pip or pipx works.

Development is hosted at Github. The latest downloads and documentation will be available at http://github.com/michaelcmartin/Ophis. If this is out-of-date, a Web search on "Ophis 6502 assembler" (without the quotation marks) should yield its page.

For Windows users, a prepackaged system made with py2exe is also available. The default Windows installer will use this. In this case, all you need to do is have ophis.exe in your path.
